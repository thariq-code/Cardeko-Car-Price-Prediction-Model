# Car Price Prediction Model

## Project Overview

This project predicts the selling price of used cars using Machine Learning. The model is trained using car details such as vehicle age, kilometers driven, mileage, engine capacity, fuel type, seller type, transmission type, and seats.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Flask

## Machine Learning Models

* Linear Regression
* Support Vector Regression (SVR)
* Decision Tree Regressor
* Random Forest Regressor
* Ridge Regression
* Lasso Regression

## Best Model

Random Forest Regressor

### Performance

* RMSE: 213890.68
* R² Score: 0.9341

## Project Files

* Project_Cardeko Car Price Prediction Model.ipynb
* app.py
* index.html
* best_model.pkl

## How to Run

1. Install required libraries.
2. Run the notebook to train the model.
3. Run app.py
4. Open http://localhost:8001 in your browser.
5. Enter car details and predict the price.

## Conclusion

Random Forest Regressor achieved the best performance and is used for car price prediction in the Flask web application.
